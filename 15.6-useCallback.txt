//! useCallback() Hook

//! What is useCallback()?
--> useCallback() is a built-in Hook introduced in React 16.8.
--> It is used to memoize (cache) a function.
--> React returns the same function reference until the dependencies change.
--> It prevents unnecessary function recreation during re-renders.

//!Definition:
--> useCallback() is a Hook that memoizes a function to improve performance by preventing unnecessary function recreation.

//! What is Function Memoization?
--> Function Memoization means storing a function in memory.
--> If the dependencies do not change, React reuses the existing function instead of creating a new one.

//! Syntax
const memoizedFunction = useCallback(() => {
    Function Logic
}, [dependencies]);

//! Import
import { useCallback } from "react";

//! Why useCallback()?
--> Prevents unnecessary function recreation.
--> Improves Performance.
--> Prevents unnecessary Child Component re-rendering.
--> Works well with React.memo().

//!Example

//*App.jsx
import React, { useCallback, useState } from 'react'
import Title from './useCallback/Title'
import Count from './useCallback/Count'
import Button from './useCallback/Button'

const App = () => {
  let [age, setAge] = useState(25)
  let [salary, setSalary] = useState(10000)
  let handleAge = useCallback(() => {
    setAge(age + 1)
  }, [age])
  let handleSalary = useCallback(() => {
    setSalary(salary + 10000)
  }, [salary])
  return (
    <div>
      <Title />
      <Count text="Age" count={age} />
      <Button fun={handleAge}>Increment Age</Button>
      <Count text="Salary" count={salary} />
      <Button fun={handleSalary}>Increment Salary</Button>
    </div>
  )
}

export default App

//*Title.jsx
import { memo } from 'react'

const Title = () => {
  console.log("Title Component Rendering....")
  return (
    <h1>useCallback Hook</h1>
  )
}

export default memo(Title)

//*Count.jsx
import { memo } from 'react'

const Count = ({ text, count }) => {
  console.log(`${text} Component Rendering...`)
  return (
    <div>
      <h1> {text} : {count}</h1>
    </div>
  )
}

export default memo(Count)

//*Button.jsx
import React, { memo } from 'react'

const Button = ({ children, fun }) => {
  console.log(`${children} Component Rendering...`)
  return (
    <button onClick={fun}>{children}</button>
  )
}

export default memo(Button)

//! Flow
Component Re-renders
         ↓
Dependencies Changed?
       /       \
     Yes        No
      ↓          ↓
Create New     Reuse Existing
Function         Function

//! Dependency Array
const handleClick = useCallback(() => {
    console.log("Clicked");
}, []);
--> Function is created only once.

------------------------------------
const handleClick = useCallback(() => {
    console.log(count);
}, [count]);
--> Function is recreated only when count changes.

//! useCallback() vs useMemo()

| useCallback() | useMemo() |
|----------------|-----------|
| Memoizes Functions | Memoizes Values |
| Returns a Function | Returns a Value |
| Used for Event Handlers | Used for Expensive Calculations |
| Prevents Function Recreation | Prevents Value Recalculation |
