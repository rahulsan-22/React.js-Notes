//! useMemo() Hook

//! What is useMemo()?
--> useMemo() is a built-in Hook introduced in React 16.8.
--> It is used to memoize the result of an expensive calculation.
--> It recalculates the value only when its dependencies change.

//!Definition:
--> useMemo() is a Hook that caches (memoizes) a computed value to avoid unnecessary recalculations and improve performance.

//! What is Memoization?
--> Memoization is an optimization technique.
--> It stores the result of a computation.
--> If the same input is received again, the cached result is returned instead of recalculating it.

Example:
2 + 3 = 5

--> The result (5) is stored.
--> Next time React returns the stored value instead of calculating again.

//! Syntax
const memoizedValue = useMemo(() => {
    return expensiveCalculation();
}, [dependencies]);

//! Import
import { useMemo } from "react";

//! Why useMemo()?
--> Improves Performance.
--> Avoids Unnecessary Calculations.

//! Example
import { useMemo, useState } from 'react'

const MemoComponent = () => {
  let [age, setAge] = useState(25)
  let [salary, setSalary] = useState(10000)
  let handleAge = () => {
    setAge(age + 1)
  }
  let handleSalary = () => {
    setSalary(salary + 10000)
  }
  let checkAgeOddorEven = useMemo(() => {
    let i = 0;
    while(i<1000000000) i++
    return age%2
  },[age])
  return (
    <div>
      <h1>Age : {age}</h1>
      <button onClick={handleAge}>Increment Age</button>
      {checkAgeOddorEven === 1 ? <mark>Odd</mark> : <mark>Even</mark>}
      <h1>Salary : {salary}</h1>
      <button onClick={handleSalary}>Increment Salary</button>
    </div>
  )
}

export default MemoComponent

//!Result:
Increment Age
--> Recalculates.

Increment Salary
--> Does NOT recalculate.

Reason:
--> count did not change.


//! Flow
Component Render
        ↓
Dependency Changed?
      /     \
    Yes      No
    ↓         ↓
Recalculate  Return Cached Value

//! Dependency Array
useMemo(() => {
    return calculate();
}, [count]);

--> Runs only when count changes.

//!Multiple Dependencies:
useMemo(() => {
    return calculate();
}, [count, age]);
--> Executes when count or age changes.
