//! Lifecycle Methods in Class Components

//! What is a Lifecycle?
--> Every React Class Component goes through different phases from creation to removal.
--> These phases are called the Component Lifecycle.

Definition:
--> Component Lifecycle is the sequence of stages that a Class Component goes through from its creation until it is removed from the DOM.


//! Lifecycle Phases
1. Mounting
2. Updating
3. Unmounting
4. Error Handling

//! 1. Mounting Phase
--> Mounting is the process of creating and displaying a Component for the first time.

//*Methods:
i) constructor()
ii) render()
iii) componentDidMount()

//! constructor()
--> Called first when the Component is created.
--> Used to initialize State and bind methods.
--> Runs only once.

//! render()
--> Mandatory method.
--> Returns JSX.
--> Runs on every change in state/props.

//! componentDidMount()
--> Executes immediately after the Component is rendered.
--> Called only once.
--> Best place for:
    --> API Calls
    --> Timers
    --> Event Listeners

//! 2. Updating Phase
--> Runs whenever State or Props change.

//*Methods:
i) render()
ii) componentDidUpdate()


//! render()
--> Executes again after State or Props update.

//! componentDidUpdate()
--> Executes after the Component updates.
--> Used for:
    --> API Calls (based on changes)
    --> DOM Updates
    --> Side Effects

//! 3. Unmounting Phase
--> Occurs when a Component is removed from the DOM.

//*Method:
componentWillUnmount()
--> Used for Cleanup.

Examples:
--> Remove Event Listeners
--> Clear Timers
--> Cancel API Requests
--> Close WebSocket Connections


//! 4. Error Handling Phase
-->Occurs after an error happens in one of its descendant (child) components.

//*Method:
componentDidCatch(error, info)
--> Catches errors in Child Components.
--> Prevents the entire application from crashing.


//! Lifecycle Methods vs Functional Components
| Class Lifecycle | Functional Component |
|-----------------|----------------------|
| componentDidMount() | useEffect(() => {}, []) |
| componentDidUpdate() | useEffect(() => {}, [dependency]) |
| componentWillUnmount() | useEffect(() => { return () => {} }, []) |
